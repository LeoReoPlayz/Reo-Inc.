Python Pacman Recreation

This project is a Python-based recreation of the classic game Pacman. Originally released in 1980, Pacman has become an iconic game in the world of arcade gaming. The Python Pacman Recreation project allows users to experience the thrill of the original game on their computers.

In Pacman, players control a yellow, circular character named Pac-Man as it moves through a maze filled with dots or pellets. The objective is to eat all the dots in the maze while avoiding four colorful ghostsâ€”Blinky, Pinky, Inky, and Clyde. The maze also contains larger, flashing dots known as power pellets, which temporarily empower Pac-Man to turn the tables and eat the ghosts. Each ghost has its own distinct behavior, making the game both challenging and strategic.

This Python-based version brings the classic gameplay to a new platform. Players can enjoy the familiar mechanics and challenges of Pacman while experiencing the project's unique Python implementation.
